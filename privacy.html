Política de Privacidade

Última atualização: 28/06/2025

1. Dados Coletados

  • IDs de usuários do Discord
  • Informações das músicas salvas em playlists

2. Uso dos Dados

Os dados são usados exclusivamente para permitir que cada usuário acesse as músicas que salvou anteriormente.

3. Proteção de Dados

Os dados são armazenados de forma segura em nosso banco de dados e não são compartilhados com terceiros.

4. Direito de Exclusão

Você pode excluir todos os seus dados a qualquer momento usando os comandos disponibilizados no bot.

5. Contato

Para dúvidas ou solicitações relacionadas a privacidade, entre em contato no servidor oficial: https://dsc.gg/judsonzinho
